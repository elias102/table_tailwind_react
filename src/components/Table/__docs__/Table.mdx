# Table

Table component with different props.

#### Example

<Canvas of={Table.Primary} />

## Usage

```ts
import {Table} from "table_tailwind_react";

const Example = () => {
  return (
      <Table
      headerCellList={headerCellList}
      dataCellList={dataCellList}
      shadowColorHeader={shadowColorHeader}
      checkboxBorderColor={checkboxBorderColor}
    />
  );
};

export default Example;
```

#### Arguments

- **headerCellList** _`() => void`_ - A string that represents the text content of the button.
- **dataCellList** - A boolean indicating whether the button should have a primary styling or not. Typically, a primary button stands out as the main action in a user interface.
- **shadowColorHeader** - A boolean indicating whether the button should be disabled or not. When disabled, the button cannot be clicked or interacted with.
- **checkboxBorderColor** - A string with one of three possible values: "small," "medium," or "large." It defines the size or dimensions of the button.
